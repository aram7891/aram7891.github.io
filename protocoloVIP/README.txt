¡Gracias por tu compra!

— ¿CÓMO USARLO? —————————————————————————————

1. Descomprime este archivo .zip en cualquier carpeta de tu computadora.
2. Abre la carpeta resultante.
3. Haz doble clic en el archivo "index.html".
4. Se abrirá en tu navegador (Chrome, Safari, Firefox, Edge — el que uses).
5. ¡Listo! Ya puedes empezar a trabajar el Protocolo.

— ¿DÓNDE SE GUARDAN MIS RESPUESTAS? ————————————

Tus notas se guardan automáticamente en tu navegador (localStorage)
para esa carpeta específica. Mientras uses el mismo navegador en la
misma computadora, tus respuestas seguirán ahí cada vez que abras el
archivo.

Si quieres conservar una copia permanente, usa el botón
"Descargar PDF" desde la barra superior.

— FUNCIONA SIN INTERNET ———————————————————————

Una vez descargado, el Protocolo funciona completamente sin conexión.
La única excepción es la primera vez que generes un PDF, que descarga
una librería pequeña desde internet.

— RECOMENDACIONES ——————————————————————————————

· Úsalo en una computadora (laptop o escritorio) para tener la mejor
  experiencia de escritura.
· Tómate tu tiempo. Este Protocolo está diseñado para acompañarte,
  no para terminarse en un día.
· Cuando termines un módulo, descarga tu PDF y guárdalo. Es tu
  registro personal del proceso.
